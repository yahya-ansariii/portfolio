Book Recommendation System

Project information

  • Category: Unsupervised ML Project
  • Client: AlmaBetter
  • Project date: June 2023
  • Project URL: Deployed link
    Github Repository

The dataset is Book Crossing dataset which consists of 3 files, Books, Users, and Ratings. Developed a book recommendation system using unsupervised learning techniques to generate personalized recommendations for users. Pre-processed and cleaned data to handle missing values and outliers, and visualized relationships between variables to gain insights. Built several recommender systems, including popularity-based filtering, correlation-based recommendations, collaborative filtering using cosine similarity and k-nearest neighbors.
